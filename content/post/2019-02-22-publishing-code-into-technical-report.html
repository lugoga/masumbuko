You are creating a report based on some analysis you are doing in R. You are pasting code snippes and figures into your report document. The document is reviewed and you got comments and feedback of something went wrong in your report and you ought to redo it all over again.

The publishing tools in Rstudio allows you to run a script and automaticaly generate document containing the results, including images of each figures generated and imported; the code chunk and code blocks with text and mathematical equations that you insert and inserting references cited.

The publishing features allow you to gnerate HTML, Latex, Word, PDF, and presentations from your code. These document can display the code itsef used to make figure or run the analysis with the plain text that describe the process.
